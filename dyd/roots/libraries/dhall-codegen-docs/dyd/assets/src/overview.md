# dhall-codegen

## About

`dhall-codegen` is a pure Dhall library that can be used to define a schema and generate equivalent definitions in multiple programming languages. The goal of this library is to make it easier to create consistent data models or API contracts across different languages and different systems.

For example, a schema:
```dhall
{{#include examples/person/schema.dhall}}
```

The generated definition in Typescript:
```ts
{{#include examples/person/typescript.ts}}
```

In Python (3.8 compatible):
```python
{{#include examples/person/python.py}}
```

In Go:
```go
{{#include examples/person/go.go}}
```

And in JSON Schema 2020-12:
```json
{{#include examples/person/json-schema-2020-12.json}}
```

## What's in this package

This package provides a core grammar for the codegen schema language plus a set of pure Dhall code renderers for different languages. Today `dhall-codegen` has renderers for:

- TypeScript (pure types or Zod validators)
- Python
- Go
- JSON Schema (draft-07 and 2020-12)
- Dhall itself


## Quick Start

Renderers are Dhall functions that take in a schema and return `Text`. You can use `dhall text` to evaluate them via command-line.

With a file defining a schema document:

```dhall
let Grammar = ./dhall-codegen/grammar.dhall

let Document = Grammar.Document

let Person = ./schemas/person.dhall

let Organization = ./schemas/organization.dhall

in  Document::{
    , headers =
      [ "// This file was generated by dhall-codegen."
      , "// Do not edit this file directly."
      ]
    , schemas = [ Person, Organization ]
    }
```

You can render that document inline from the CLI:

```bash
dhall text \
  --expression 'let r = ./dhall-codegen/renderer-typescript/package.dhall
               in  r.render r.options::{=} (./my-document.dhall)'
```

Or you can render inline another file:

```dhall
let TypeScript = ./dhall-codegen/renderer-typescript/package.dhall

let document = ./my-document.dhall

in  TypeScript.render TypeScript.options::{=} document
```

Then evaluate it with:

```bash
dhall text --file ./ts-output.dhall --output ./generated.ts
```
